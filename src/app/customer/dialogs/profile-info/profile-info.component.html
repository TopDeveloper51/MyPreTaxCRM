Refund Status

Sales Process Status = Cancel-Refund -> "Refund Issued"
Current Refund Request = TRUE -> "Refund Request"
if none of the above -> "None"

Payment Status

derived based on Sales Process Status
if exactly Annual 2017 or Annual 2018 -> not clear,
if contains w/f pymnt. -> not paid at all,
if contains Paid Partially -> partially paid,
if exactly Annual 2017 New, Annual 2017 Renew, Annual 2018 New, Annual 2018 Renew -> fully paid
if none of the above -> "None" -> not applicable
